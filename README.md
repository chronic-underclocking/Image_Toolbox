# Image Toolbox
 An image manipulation program written in Java.
 
  ![Interface](https://i.ibb.co/J7jMHrT/GUI.png)
 
 # Features:
 
 **- Basic Tools:**
    
    Crop
    
    Rotate
    
    Mirror
    
    Resize
 
 **- Filters:**
    
    Black and White
    
    Edge Detection
    
    Smooth
    
    Sharp
    
    ...more to be added
    
 **- ASCII Art generator:**
 
    Transforms an image to ASCII art
 
   ![Emma Stone drawn in ASCII](https://i.ibb.co/KGv26kr/notWeird.png)
