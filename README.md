# Image Toolbox
 An image manipulation program written in Java.
 
 # Features:
 
 **- Basic Tools:**
    Crop
    Rotate
    Mirror
    Resize
 
 **- Filters:**
    Black and White
    Edge Detection
    Smooth
    Sharp
    ...more to be added
    
 **- ASCII Art generator:**
    Transforms an image to ASCII art
    ![Emma Stone drawn in ASCII](https://ibb.co/LP7xSWZ)
